Teknologi yang digunakan adalah HTML, CSS, javascript, mysql, dan php laravel sebagai backend
tambahan aplikasi yang dibutuhkan untuk menggunakannya adalah xampp

Petunjuk Penggunaan:
- setelah berhasil clone, buka XAMPP kemudian aktifkan mysql dan apache
- buka chrome, kemudian ketik link http://localhost/phpmyadmin
- buat database yang bernama budgetapp
- buka terminal, lalu arahkan direktori pada folder api, lalu ketik command "php artisan migrate"
- aktifkan rest server dengan perintah "php artisan serve"
- untuk membuka websitenya, buka folder html kemudian klik file index.html
- di website tersebut, terdapat 2 menu yang bisa digunakan yaitu Home dan Harian.
- pada menu Home dapat menambahkan data tracking budget, menghapus data dan menampilkan seluruh informasi
- pada menu Harian, hanya menampilkan pengeluaran, pemasukkan pada hari ini(hari sedang menggunakannya).
